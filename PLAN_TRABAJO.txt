3. PASOS PARA MIGRAR A POSTGRESQL:

   a) En package.json, agregar prisma/postgresql:
      npm install @prisma/client pg
      npm install prisma --save-dev
   
   b) Cambiar prisma/schema.prisma:
      datasource db {
        provider = "postgresql"
        url      = env("DATABASE_URL")
      }
   
   c) Crear archivo .env.production:
      DATABASE_URL="postgresql://user:pass@host:5432/db"
   
   d) Ejecutar migración:
      npx prisma db push
      npx prisma generate
